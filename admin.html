🗺️ 数据采集中心 · 高德POI导入说明
▼ 展开

功能定位

「数据采集中心」是一个基于高德地图 Web 服务 API 的 POI 批量检索 + 一键导入工具，用于将集市、早市、夜市等相关地点的真实数据快速导入到集市审核队列，由管理员审核后上线。

使用前提

  • 需要一个高德开放平台 Web 服务密钥（注意：不是 JS API 密钥，两者不同）
  • 申请地址：https://lbs.amap.com/ → 控制台 → 创建应用 → 添加 Key → 服务平台选 "Web 服务"
  • 密钥填写后会自动保存在本地浏览器，下次打开无需重填

操作步骤

  1. 进入「数据采集中心」页面，填入高德 Web 服务 API Key 并保存
  2. 在搜索框输入关键词（如：集市、早市、夜市），选择城市（可留空=全国）
  3. 点击「搜索」或使用快速关键词按钮，查看搜索结果卡片
  4. 勾选需要导入的 POI 数据，点击「导入选中」
  5. 导入后进入「采集数据库」页面可查看队列，支持批量审核通过

数据字段说明

高德字段 对应集市字段 备注
name集市名称原始 POI 名
location经度/纬度格式 "lng,lat"
address详细地址—
pname/cityname/adname省/市/区自动填入地区
id来源标识amap_{id}，防重复导入

后续计划

当自研爬虫程序开发完成后，可通过对比 amap_{id} 标识与爬虫采集数据进行去重合并，将高质量的爬虫数据（如时间表、摊位信息、照片等）追加到已导入的高德 POI 记录上，形成完整的集市档案。

⚠️ 注意： Web 服务密钥与前端地图 JS API 密钥是两套不同的密钥，请勿混用。Web 服务密钥用于后端 HTTP 请求，不可暴露到前端页面代码中，本系统通过后端代理调用保证安全。
